## 🔄 Fluxo de Atualização Semanal

Para atualizar os dados do playbook, siga os passos abaixo:

1. Acesse o arquivo [`dados.js`](https://github.com/universocsdigital-wq/icbid/blob/main/dados.js) neste repositório.
2. Clique no **ícone de lápis** (Editar) no canto superior direito do arquivo.
3. Altere apenas os valores numéricos ou textos entre aspas (⚠️ **Não mude os nomes das variáveis**).
4. Clique no botão verde **Commit changes** para salvar.
5. Aguarde cerca de **1 minuto** para que o painel seja atualizado automaticamente na URL acima.
